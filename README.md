# I'm Sorry… From My Heart ❤️

A cinematic apology web experience built with emotion, animation, and interactive storytelling 💔➡️❤️

---

## 💖 About This Project

This is not just a website… it's an **experience**.

Designed to express feelings in a way words alone cannot, this project takes the user through a **6-slide emotional journey** — from regret to hope.

Perfect for:
- Saying sorry in a unique way
- Making someone feel special
- Turning emotions into something beautiful

---

## 🎬 Features

- 💔 6 Cinematic Slides (Story-based flow)
- 🎭 Smooth Animations (Framer Motion)
- 🎵 Background Music (Emotional feel)
- 🖼 Memory Section (GIF / Images)
- 🎁 Interactive CTA Ending
- ✨ Touch / Click Effects
- 📱 Fully Responsive Design

---

## 🧠 Flow of Experience

1. Silence → Realization  
2. Guilt → Memories  
3. Apology → Hope ❤️  

Each slide builds emotion and leads to a meaningful ending.

---

## 🛠 Tech Stack

- ⚛️ Next.js – React Framework for building fast UI
- 🎨 Tailwind CSS – Modern and responsive styling
- 🎞️ Framer Motion – Smooth animations & transitions
- 🎊 Canvas Confetti – Celebration effect on final slide
- 🖼️ Swiper.js – Memory slideshow (if used)

---

## 🖥 Local Setup

To run this project locally:

```bash
# Clone the repository
git clone https://github.com/manorath-rastogi/sorry-from-my-heart.git

# Navigate into the folder
cd sorry-from-my-heart

# Install dependencies
npm install

# Start development server
npm run dev
```

Then open:
👉 http://localhost:3000
